▸ CallMonitor(options)
src/fon.js

Receive events for incoming and outgoing calls.

Enable the callmonitor on your Fritz!Box by dialing #96*5* and disable with #96*4* Set options.callmonitorport to 1012 or a custom port.

This is an implementation of Thorsten Basse's Fritz!Box Call Monitor

CallMonitor(options: Object): EventEmitter
Parameters
options (Object) FritzBox.js options object.
Returns
EventEmitter: Returns an event emitter for you to catch events with.
